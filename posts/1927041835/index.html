我的Hexo博客第一篇博文

发表于 2018-04-10 | 分类于 Hexo |

说实话，没有想到 Hexo 的部署这么另类，折腾了近两天才算是彻底的搞定了，也成功的绑定到 hexo.imydl.tech 这个二级域名了，虽然现在还没有办法实现 HTTPS （好像托管在 Git 上的目前没有很好的办法，继续学习研究中……）。

说实话，明月还没有体验过纯静态化的博客站点呢，习惯了动态化博客平台 WordPress 、 Typecho 这些后突然使用一下 Hexo 还真是不习惯呀，不过 Hexo 撰写博客的方式真的好另类呀！竟然可以在 Linux 下使用 Vim 来直接完成，还是支持 MarkDown 语法的，这个真心是好赞呀！以后学习 Vim 也方便了，哈哈！

剩下的就是如何实现 HTTPS 了，已经使用了 Next 这个 Hexo 主题了，效果很满意，就是需要自己慢慢再研究了，就是这个 HTTPS 目前毫无头绪的感觉， imydl.tech 已经加入 HSTS 的预加载表了，目前在 Chrome 还可以浏览，在 Firefox 竟然是直接无法访问，必须是 HTTPS 的才可以，没有想到自己给自己挖了一个坑，郁闷呀！

好了，第一篇博文就不长篇大论了，主要是为了体验一下 Hexo 发博文的快感:-)，体验非常的好，继续去学习消化 Hexo 以及如何实现 HTTPS 了。
