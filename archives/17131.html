本博客正式托管于Coding方便国内用户访问

发表于 2018-04-13 | 分类于 news | | 阅读次数:

本来本博客是托管在 pages.github.io 上的，但是众做周知的原因 GitHub 国内访问的速度体验真心是不敢恭维呀，并且pages.github.io还不支持HTTPS，需要借助Nginx反向代理来实现HTTPS，这个又再次延长了博客访问的速度，还是习惯将博客站点放在国内的体验呀。

还好 Hexo 支持双托管好像，只需要在 Hexo 站点配置文件_config.yml的托管推送（deploy）配置位置添加一个Coding的项目地址即可实现 GitHub 和 Coding 双托管推送了。

只需要将上述的配置代码里的xxxx改为自己对应的项目名称和链接即可，这样就可以实现 GitHub 和 Coding 双托管推送了。

将Coding解析给国内线路，GitHub的解析给海外线路，科学合理的分配一下DNS解析线路即可，很科学、很高效外加双保险！是不是很赞呀！嘿嘿。
